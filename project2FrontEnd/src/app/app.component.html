Test Automator by Cybernate

by Paul Cappelan, Brian Maguire, Jacob Golding, and Henry Simon

Test Automator provides on-demand functionality testing and feedback reports for Revature's Assignforce application.
Because of Test Automator's modular architecture, developers can easily modify it to test any existent web application.

The technologies used in this project include:
Angular v6, Java MVC, TestNG, Cucumber, Gherkin, Selenium, Apache Maven, Node.js, Protractor, Jasmine, Karma, Jenkins,
Apache Tomcat, AWS EC2 + RDS, Oracle, and Git/Github.
